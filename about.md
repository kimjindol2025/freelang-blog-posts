# FreeLang 기술 블로그

이 블로그는 **FreeLang** 팀의 기술 글을 공유하는 플랫폼입니다.

## 주요 주제

- **분산 시스템**: Raft 합의, Vector Clock, Quorum Locking
- **데이터베이스 최적화**: Zero-Copy, LSM Tree, B-Tree
- **성능 튜닝**: SIMD, Cache Line, CPU 스케줄링
- **저수준 프로그래밍**: Go, Rust의 메모리 모델과 동시성
- **시스템 엔지니어링**: Kubernetes, Docker, CI/CD, 모니터링

## 컨텐츠 구성

- **Phase 1-2**: 기초부터 심화까지의 기술 개념 (10개 포스트)
- **Phase 3**: 고급 시스템 엔지니어링 (20개 포스트)
- **Phase 4**: 심화된 기술 주제 (15개 포스트)
